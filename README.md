# 사업 관리 앱

사업 기본정보, 일정, 메모를 한 곳에서 관리하는 단일 HTML 파일 앱입니다.

## 기능

- **대시보드** — 진행중 사업, 30일 내 일정, 기한 초과 현황 한눈에 보기
- **사업 목록** — 사업 추가/수정/삭제, 분류·상태·예산·담당자·메모 관리
- **일정 관리** — 목록 뷰 / 캘린더 뷰 전환, D-day 자동 계산, 완료 처리
- **메모장** — 자유롭게 메모 기록, 카드 형태로 보기

## 데이터 저장

`localStorage`를 사용해 브라우저에 자동 저장됩니다.  
같은 브라우저에서 열면 데이터가 유지됩니다.

## 사용 방법

### 로컬에서 열기
```
index.html 파일을 브라우저에서 직접 열기
```

### GitHub Pages로 호스팅
1. 이 레포를 GitHub에 push
2. Settings → Pages → Branch: main, folder: / (root) 선택
3. `https://{username}.github.io/{repo-name}/` 으로 접속

## 파일 구조

```
index.html   ← 전부 여기 있습니다 (HTML + CSS + JS)
README.md
```
